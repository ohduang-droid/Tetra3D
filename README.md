# Tetra3D

3D 版的俄罗斯方块，使用 Three.js 渲染与 OrbitControls 控制。支持鼠标旋转、缩放、平移视角，并提供经典的方块操作、消行与计分逻辑。

## 运行方式
1. 安装一个本地静态服务器（任选其一）：
   - 使用 Node.js: `npm install -g http-server`
   - 或使用 Python 3 自带: `python -m http.server 8000`
2. 在仓库根目录启动服务器，然后浏览器访问 `http://localhost:8000`。

> 浏览器需支持 ES modules。项目使用 CDN 版 Three.js，无需额外构建步骤。

## 操作说明
- 鼠标：拖拽旋转，滚轮缩放，按住 Shift 拖拽以平移。
- 键盘：`←`/`→` 左右移动，`↑` 旋转，`↓` 加速下落。
- 按钮：开始、暂停、重新开始。

## 功能特性
- 7 种标准方块（I, O, T, S, Z, L, J）随机生成与旋转。
- 碰撞检测、落地锁定、完整行检测与消行动画（淡出+缩放）。
- 计分与累计消行统计，HUD 状态显示。
- 游戏状态控制：开始、暂停、重新开始；超出顶部即结束。
- 3D 棋盘与光照，搭配 OrbitControls 提供自由视角。
